# 研究报告：极简画廊卡片布局

## 决策：极简化卡片 UI

### 方案：移除“干扰性”信息
- **具体选择**：将卡片直接降级为“图片 + 极小标题 + 标签”模式。
- **技术理由**：通过减少文本元素，能够极大缩短卡片总高度，解决在多列布局下文字占比过大的问题。

## 决策：样式调整
- **具体选择**：将标题字体从 `text-xl` 调整为 `text-base` 或 `text-lg`，内边距从 `p-6` 压缩到 `p-4`，移除原本为了“复制”按钮预留的 flex 空间。
- **技术理由**：5 列布局下单张卡片宽度有限，大字体和宽边距会使文本区域显得更加局促。
