# UI 布局契约：移动端适配 (Mobile UI Adaptation)

## 响应式策略定义

系统遵循 **Mobile First (移动优先)** 策略。默认样式针对小屏幕，通过 Tailwind 的 `sm:`, `md:`, `lg:` 前缀应用桌面端增强。

### 1. 导航栏 (Navbar)
| 元素 | 移动端 (< 640px) | 桌面端 (> 640px) |
| :--- | :--- | :--- |
| **高度** | 72px (`h-[72px]`) | 80px (`h-20`) |
| **Logo** | 仅图标或精简文字 | 完整名称 + 标语 |
| **主题切换** | 仅显示图标按钮 | 显示当前主题名称 + 箭头 |
| **投稿按钮** | 仅显示 "+" 图标 | 显示 "+ 我要投稿" 文字 |

### 2. 画廊网格 (Gallery Grid)
| 断点 | 列数 (`grid-cols-N`) | 卡片内边距 |
| :--- | :--- | :--- |
| **默认 (< 640px)** | `grid-cols-2` | 12px (`p-3`) |
| **平板 (> 768px)** | `grid-cols-3` | 16px (`p-4`) |
| **桌面 (> 1024px)** | `grid-cols-4` | 16px (`p-4`) |
| **超大屏 (> 1280px)** | `grid-cols-5` | 16px (`p-4`) |

### 3. 弹窗组件 (Modals)
| 状态 | 移动端 | 桌面端 |
| :--- | :--- | :--- |
| **圆角** | 仅顶部圆角 (`rounded-t-3xl`) | 全圆角 (`rounded-[3rem]`) |
| **尺寸** | 宽 100%, 高 92%-95% | `max-w-6xl` 或 `max-w-4xl` |
| **图片高度** | **Max 35vh (`max-h-[35vh]`)** | 自动比例 |
| **内容布局** | 垂直堆叠 (Media 上, Form 下) | 水平排列 (Media 左, Form 右) |

### 4. 全屏预览 (Full Screen Lightbox)
- **触发**: 点击详情页中的任何图片/视频媒体。
- **表现**: 弹出全屏覆盖层，背景 90% 黑色，媒体以 `object-contain` 模式撑满屏幕。
- **交互**: 点击背景或媒体任意位置即可退出。

## 交互准则
- **点击目标**: 最小交互区域需达到 `44x44 px`。
- **字体大小**: 正文不低于 `14px`，辅助文字不低于 `10px`。
- **滚动**: 全屏弹窗内部需支持独立滚动，且背景层锁定（`overflow: hidden`）。
